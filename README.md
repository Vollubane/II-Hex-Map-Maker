# iiHexMapMaker

## AI usage

This project may use AI-assisted tools **only** in the following cases:

- writing or helping to write **comments** in the code;
- **development** assistance when it involves **mathematical functions** (formulas, geometry, coordinate conversions, etc.);
- **formatting** and **proofreading** corrections;
- other minor tasks (e.g. finding documentation).

**No** AI is used for **artistic creation** (art direction, design, etc.).

**No** **image-generating** or **3D asset** artificial intelligence has been used for this project, **nor will it be**.
